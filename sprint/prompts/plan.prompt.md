# Sprint — plan mode

Prepare `work/sprint-{id}/plan.md` before the sprint starts.

Read [SKILL.md](../SKILL.md).

## Context

[Sprint id/dates, docs/product/backlog.md, `work/{epic}/tasks.md` and design.md for epics in scope, roadmap]

## Steps

1. Sprint goal (one sentence)
2. Scope — epics in focus with Epic ID, title, and `work/{epic}/` paths
3. Committed tasks from each epic's `tasks.md`
4. Capacity, dependencies, risks, out of scope, sprint DoD
